Набор тестов, проверяющий функциональность домашних страниц сайтов google.com, it-academy.by, tut.by

google.com 
 * открыть поисковик, в строку поиска ввести слово Java
 * убедиться, что название страницы с результатом поиска содержит слово Java

it-academy.by
 * раздел меню "На кого учиться": прочитать и вывести на консоль все пункты выпадающего меню и подменю пунктов при наличии
 * раздел "Каталог курсов": найти пункт "Маркетинг" и перейти по этой ссылке

tut.by
 * Прочитать с сайта название главной новости.
 * Клинуть на главную новость и подсчитать количество параграфов в тексте новости.

Три тестовых класса включают 10 тестов. Ход выполнения программы логируется в консоль и лог-файл, отчеты surefire создаются. Порядок выполнения тестов задается в testNG.xml
